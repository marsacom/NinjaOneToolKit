## NinjaOneCompareDevices

This script uses the NinjaOne API to perform a variety of functions. With this script you can...

1. List all devices in NinjaOne
2. List all devices in the Domain 
3. List devices missing from NinjaOne & Domain
4. Add computer to Domain
5. Add computer to NinjaOne

This script also uses openpyxl to read/write to XLSX file of computers as the control for comparing to NinjaOne and the Domain.

Be sure to specify your CLIENT_ID, CLIENT_SECRET, XL_PATH, & LOG_PATH in .env
